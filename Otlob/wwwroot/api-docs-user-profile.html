User Profile API

Documentation for mobile app integration

Base URL: https://otlob.runasp.net
Authentication Required: All endpoints require a Bearer Token in the Authorization header.

GET /api/UserProfiles/profile

Retrieves the current user's profile information.

Response (JSON)
{
  "isSuccess": true,
  "apiError": { "code": "", "description": "", "statusCode": null },
  "value": {
    "firstName": "John",
    "lastName": "Doe",
    "email": "user@example.com",
    "userName": "username",
    "phoneNumber": "010...",
    "gender": 0,
    "dateOfBirth": "1990-01-01",
    "image": "image.jpg"
  }
}
Failure Response (Example)
{
  "type": "https://tools.ietf.org/html/rfc7231#section-6.5.1",
  "title": "Bad Request",
  "status": 400,
  "detail": "User not found.",
  "code": "UserProfile.UserNotFound"
}

PUT /api/UserProfiles/update

Updates user profile data.

Request Body (JSON)
            {
              "firstName": "string",
              "lastName": "string",
              "email": "string",
              "userName": "string",
              "phoneNumber": "string",
              "gender": 0,
              "dateOfBirth": "yyyy-MM-dd"
            }

PUT /api/UserProfiles/update-profile-picture

Updates profile image (Multipart Form Data).

Request

Parameter name: image (File upload)

DELETE /api/UserProfiles/delete

Deletes user account.
